# Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`

# Truffle deploy

## Compile Contracts

`newtruffle compile`

## Run Tests

`newtruffle test ./truffle-test/*`

## deploy devnet
`newtruffle deploy --network devnet`
